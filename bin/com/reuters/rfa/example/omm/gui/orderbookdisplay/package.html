OrderBookDisplay is a GUI application that visually demonstrates the MarketByOrder feature. This GUI demo application is a multi-threaded application consuming real-time, level 2 market data, using the RFA Java API. It displays all the market orders for a given item. Additionally, it displays the summary data, and enables the user to sort the BID or ASK prices.

This application does the following:
  • Creates a GUI
  • Accepts an Item and service name entered via GUI
  • Displays level 2 market data for the selected item (BID and ASK orders)
  • Dynamically updates the dispaly
  • Closes the application

The OrderBookDisplay application currently does not support command line arguments.

The application uses session named "myNamespace::OBSession". The session must be configured.

You can run this example as an application with the following commands:

    java com.reuters.rfa.example.omm.gui.OrderBookDisplay
