Contains a framework utilities for consumer/subscriber application to process various IDN data models, including TS1 and chains of reference pages. These utilities can be used in conjunction with the {@linkplain com.reuters.rfa.example.framework.sub Sub Framework}.  TS1TimeSeries is the primary class of the TS1 decoder.  RefChain is an abstract class for processing QQ page sequences.  RefChainTimeSeriesDb is a RefChain implementation for process the TS1 data definitions.
