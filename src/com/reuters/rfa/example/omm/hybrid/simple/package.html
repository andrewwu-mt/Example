The following contains a simple hybrid application, which is both an OMM consumer and an OMM provider.

The {@link com.reuters.rfa.example.omm.hybrid.simple.SimpleHybridDemo SimpleHybridDemo} is intended to demonstrate how to write an application that is both an OMM consumer and an OMM provider. SimpleHybridDemo's provider side listens to a request, and passes the request to its OMM consumer side. Every time it receives a {@link com.reuters.rfa.rdm.RDMMsgTypes LOGIN} request, the OMM consumer side creates {@link com.reuters.rfa.session.omm.OMMConsumer OMMConsumer}.

SimpleHybridDemo currently supports the following command line arguments:

  • runTime: Sets the number of seconds for which the application will run. The default is 600.
  • session: The session must contain at least two connections (one RSSL and one RSSL_PROV). The default is myNamespace::hybridSession.
  • rdmFieldDictionary: Sets the RDMField dictionary name and location. The default is /var/triarch/RDMFieldDictionary.
  • enumType: Sets the RDMEnum dictionary name and location. The default is /var/triarch/enumType.def.
  • useReencoder: Sets whether to re-encode messages before passing them through the source/sink application. The default is true.
  • useEncodeData: Enables {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder#setEncodeData(boolean) useEncodeData} in {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder OMMMsgReencoder} when re-encoding a message. The default is false.
  • useEncodeString: Enables {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder#setEncodeString(boolean) useEncodeString} in {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder OMMMsgReencoder} when re-encoding a message. The default is false.
  • usePreEncodeDataDefs: Enables {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder#setPreEncodeDataDefs(boolean) usePreEncodeDataDefs} in {@link com.reuters.rfa.example.omm.hybrid.OMMMsgReencoder OMMMsgReencoder} when re-encoding a message. The default is false.

You can run the SimpleHybridDemo as an application by using the following command:

    java com.reuters.rfa.example.omm.hybrid.simple.SimpleHybridDemo
