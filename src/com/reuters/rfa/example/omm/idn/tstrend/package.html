AWT-based GUI that shows how to retrieve TS1 historical data from a remote service, decode it, and present it in the form of line graphs which can be seen as trends for the corresponding market activities. The example allows a user to input the item name, the periodicity (interval between samples, e.g. daily) and the number of samples for the series.  It is based on a similar example originally written for JSFC.

The application first uses the {@linkplain com.reuters.rfa.example.framework.sub framework.sub} for context initialization.  The application then loads the TS1 reference dictionary with the example framework class {@link com.reuters.rfa.example.framework.idn.RefChainTimeSeriesDefDb RefChainTimeSeriesDefDb}.  TS1 data cannot be requested until the time series definition database is complete.  When data is requested, the examples uses the example framework class {@link com.reuters.rfa.example.framework.idn.TS1TimeSeries TS1TimeSeries} to retrieve the TS1 data from a {@link com.reuters.rfa.session.Session Session} and to parse the data with the {@linkplain com.reuters.ts1 TS1 parser}.  After the time series has been retrieved, the data is scaled using the RFA class {@link com.reuters.ts1.TS1Computation TS1Computation} and graphed using AWT.

The following command line configuration parameters are currently available.

  • help - Display help information and exit. The default is false.
  • session - Consumer session. The default is myNamespace::mySession.
  • serviceName - Service name to request the preload items. The default is "IDN_RDF".
  • user - DACS username for login. The default is the value from "user.name" property or "guest".
  • position - DACS position for login. The default is local host or "1.1.1.1/net".
  • application - DACS application ID for login. The default is "256".
  • fileDictionary - load dictionary from file. The default is false.
  • enumType - enumtype.def filename. The default is "/var/triarch/enumtype.def".
  • debug - enable debug tracing. The default is false.

The following parameter can also be used.

  • rdmFieldDictionary - RDMFieldDictionary filename. The default is "/var/triarch/RDMFieldDictionary".

The TimeSeriesTrend application can be run with the following commands:

  • To retrieve real-time page using OMM consumer model
    java com.reuters.rfa.example.idn.tstrend.TimeSeriesTrend -session myNamespace::rsslSession -serviceName IDN_RDF
