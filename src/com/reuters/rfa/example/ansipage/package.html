Demonstrates the use of {@linkplain com.reuters.rfa.example.ansipage ANSIPage API} in a simple example. In this example, the Page interface is used to demonstrate how ANSI update strings are encoded from and decoded into Page objects and their respective PageUpdate lists.

The AnsiPageDemo class demonstrates how the ANSIPage API can be used to do the following:
  • Encode ANSI strings into Page objects
  • Decode ANSI strings from Page objects
  • Encode and decode PageUpdate lists

The example can be run as an application using the following command:

java com.reuters.rfa.example.ansipage.AnsiPageDemo
