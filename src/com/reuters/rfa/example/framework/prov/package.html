Contains a framework for a publisher application. PubAppContext is the main component of this framework. It performs tasks that most publisher needs to do before it can public data.

This package contain ProvDomainMgr that used for creating specific domain manager to publish OMM data with this publisher framework.

Configurable parameters are :
  • debug - enable debug tracing. The default is false.
  • provSession - session name to use. The default is "myNamespace::provSession".
    • rdmFieldDictionary - RDMFieldDictionary path and filename. The default is "/var/triarch/RDMFieldDictionary".
  • enumType - enumtype.def filename. The default is "/var/triarch/enumtype.def".
  • listenenerName - listener name for the session. The default is "".
  • acceptSessions - accept all sessions. The default is true.
  • vendor - vendor name. The default is "RFAExample".
  • pubServiceName - service name used by provider. The default is "MY_SERVICE".
  • isSource - is this original source of data. The default is false.
  • runTime - how long the application should run before exiting (in seconds). The default is -1 (infinite).
